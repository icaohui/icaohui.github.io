约瑟夫环问题

2019-05-29 数据结构预算法

问题描述

已知n个人围坐在一张圆桌周围，从编号为k的人开始报数，数到m的那个人出列；他的下一个人又从1开始报数，数到m的那个人又出列，以此类推重复下去，知道圆桌周围的人剩下最后一个，求最后剩下的那个人的初始编号。

数组解法

改问题可以理解成一个数组，用代码表示如下：

function findKing($n, $m) {
  $num = 0;
  $arr = range(1, $n);
  while(count($arr) != 1){
      foreach($arr as $k => $v){
          if($num == 2){
              unset($arr[$k]);
              $num = 0;
          }else{
              $num++;
          }
      }
  }
}

数学解决法

利用数学分析，可以发现其中的规律，用公式表达为：

f(N,M)=(f(N−1,M)+M)%N

用代码实现如下：

function findKing($n, $m) {
    $r = 0;
    for ($i = 2; $i <= $n; $i++) {
        $r = ($r + $m) % $i;
    }
    return $r + 1;
}

单向循环链表解法
